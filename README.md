## Voici les commandes à suivre pour executer le frontend

Dans votre terminal :

- Cloner le dépôt :
```bash
git clone https://github.com/Nikogjj/motus_frontend.git
```
- Se déplacer dans le dossier :
```bash
cd motus_frontend
```
- Installer les dépendances Node.js : 
```bash
npm install
```
- Lancer le serveur :
```bash
ng serve
```
